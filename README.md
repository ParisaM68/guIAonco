# guIAonco

guIAonco is a bilingual English/Spanish prototype web app that uses an AI-style interface to support healthcare workers caring for patients with breast cancer concerns. It includes learning modules on warning signs, risk factors, clinical breast exam basics, referral steps, patient education, short practice cases with feedback, and a geolocation-based clinic search.

The pilot version is designed for Peru and uses colors inspired by Peru's flag. It could be adapted to other Spanish-speaking settings with updated content, local referral pathways, and colors that reflect the country or institution.

## How to Use

Open `index.html` in a browser, or publish the project with GitHub Pages and open the public link.

The app includes:

- Learn
- Practice Cases
- Patient Education
- Clinics
- Referral
- Ask Expert

## Important Note

This is a preliminary school project. The clinical content, translations, referral guidance, and patient-facing language should be reviewed and confirmed by qualified breast cancer, public health, and Spanish-language experts before any real-world use.
